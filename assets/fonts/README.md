# Required fonts

Download and place these two files here before running the app:

1. DMSerifDisplay-Regular.ttf
   https://fonts.google.com/specimen/DM+Serif+Display

2. Outfit-VariableFont_wght.ttf
   https://fonts.google.com/specimen/Outfit

These are loaded in App.tsx via Font.loadAsync().
The app will crash at startup if these files are missing.
